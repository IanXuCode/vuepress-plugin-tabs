## Install
::: warning Note 
This plugin requires VuePress >= 1.0.0, for now you can try it via yarn add vuepress@next -D 
:::

```
yarn add @sueprbiger/vuepress-plugin-tabs -D
```

## Usage
```
// .vuepress/config.js
module.exports = {
  plugins: [
    '@superbiger/vuepress-plugin-tabs'
  ]
}
```
